Игра "Шапка" - это игра направленная на развитие логического мышления,
  развите памяти и способности составлять алгоримы.

Для запуска требуется Python, при написании использована версия 3.11
  и библиотека Pygame, при написании использована версия 2.5.2.

Для запуска игры требуется запустить фаил main.py
